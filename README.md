# MLB Rules ChatBot

A retrieval based chatbot designed to take user queries and answer them with relevant major leage rules with citations

## Overview

The chatbot processes MLB rules data into sctructured chunks, uses TF-IDF vectorization with cosine similarity to retrieve the most relevant rule.

## How to run

1. Install dependencies: pip install -r requirements.txt
2. Run the chatbot from the project root: 'python scripts/cli_chat.py'

## notes
- prebuilt model files are included in the 'models' directory, so no setup or training required
